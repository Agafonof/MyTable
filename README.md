## MyTable
Pet-проект с кастомной универсальной таблицей

---
Инструкция для запуска проекта: 

- установка зависимостей **npm i**
- запуск **npm run dev**
<br/>
Проект будет запущен на порту localhost:3000
<br/>
<br/>

<br/>

- JavaScript

- React
- Tailwind CSS
